## 🚀 Iniciando <movies>

Para iniciar o sistema, siga estas etapas:

* Configure o arquivo .env e docker-compose.yml para conexão correta com o banco de dados


* Inicie o docker com:
```
docker-compose up
```

* Inicie o servidor via yarn:
```
yarn dev
```
